# Snippets 写

> ⟦ 様式 ⟧ — how code is written in the style. Not libraries — *examples*:
> block frames, section rules, and meaning-marks (← → ⚠ ↺ ✗ ◉) shown on
> real code, per language. See, don't read.

## Format
```
<lang>/<topic>.<ext>   a short, real example commented the yoshiki way
<lang>/README.md       the conventions specific to that language
```

## Rule
Each snippet demonstrates the comment canon from `../../canon/lexicon/comments.md`
on actual code: one heavy frame per file, light rules, 5–10 meaning-marks,
nothing else. Languages first in line: rust · python · shell · lua.
