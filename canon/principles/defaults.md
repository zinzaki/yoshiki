The opinionated fallback for every common decision. A strong default, overridable
when a task says otherwise — opinionated by default, flexible underneath.

| Unspecified… | Default |
|---|---|
| density | air — generous margins, one accent per screen |
| accent color | gold as a line / edge / glyph — never a fill |
| trigger (scarlet · moss) | none — live in tone |
| background / dark | warm lacquer, never grey |
| text | bone, never pure white |
| geometry | soft 16–24px radius, optionally one sharp corner |
| block weight | at the top (the heading holds); the bottom breathes |
| frame | closed and soft; the open "break" is a deliberate device |
| motion | a breathing tempo; spinner ~90ms; no dizzy whirl |
| progress | smooth sub-symbol or dot-matrix, never a slab bar |
| glass / effect | light, matte, by place — never Apple-glossy |
| typography | serif for display · mono for data and code |
| kanji / glyph | latin for names; a kanji only when it carries weight |

Each row is the lowest rung of [`priority.md`](priority.md) made concrete: air
(1), omission (2), warmth (4), rarity (5), meaning (7).
