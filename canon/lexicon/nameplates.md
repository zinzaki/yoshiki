# Nameplates

Spaced capitals between paired marks. The pattern: `<mark> L E T T E R S <mark>`
— wide tracking turns a name into an emblem.

## Canon (current)

```
·  A  U  R  O  R  A  ·             quiet dot — default
—  A  U  R  O  R  A  —             dash — formal
⟦  A  U  R  O  R  A  ⟧             brackets — technical, HUD
◇  A  U  R  O  R  A  ◇             diamond — ceremonial
◸  A  U  R  O  R  A  ◿             corners — framed, poster-like
```

## Typeface variants (Unicode styles)

| Style | Sample | Use |
|---|---|---|
| mono | `𝙰 𝚄 𝚁 𝙾 𝚁 𝙰` | terminals, tech contexts |
| bold sans | `𝗔 𝗨 𝗥 𝗢 𝗥 𝗔` | chat display names |
| plain caps | `A U R O R A` | maximum compatibility |

## Accent (optional, dosed)

A single accent glyph after the name can add a mood — never more than one,
and only where it earns its place:

```
◇  A U R O R A  ✦  ◇          a star — signature
⟦  A U R O R A  ◆  ⟧          a mark — emphasis
```

## Rules

1. Two spaces between letters; a hair-space or thin space also works where
   width is precious.
2. Marks always come in pairs.
3. Unicode "fonts" (𝗯𝗼𝗹𝗱/𝚖𝚘𝚗𝚘) break screen readers and search — use them
   only in display contexts (profile names, banners), never in body text.
4. One nameplate per surface. It is a seal, not a watermark.
