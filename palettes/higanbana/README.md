# Higanbana 彼岸花 — spider lily · v1.0

Urushi's casting, but the elements bloom in spider-lily scarlet
(OKLCH hue ≈ 25°):

- petal `#E88479` — active-nav text, soft accents
- glow `#E65F57` — hover, progress head, cursor
- lily `#DE3C3A` — primary fills, statusbar mode
- depth `#89292C` — pressed · wash `#3A1616`

Gold stays veins-only. Signature detail: every 4th VU-meter bar is gold —
a vein in the scarlet field; the scrollbar is red lacquer `#46211C`.

Trade-off, accepted consciously: red becomes an area, so the
"rarity = power" rule is spent here. The trigger-scarlet `#BB3C33`
stays darker and duller than the lily so critical events remain distinct.
